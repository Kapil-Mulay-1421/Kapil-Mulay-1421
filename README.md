# Hi 👋, I'm Kapil Mulay
I build software across several layers, from modern ML and full-stack web applications to embedded systems and processor design.

I've built a generative AI application, a cloud-native IoT platform, a deployment-ready e-commerce website, and more. 

I love participating in hackathons and competitions, having previously represented IIT Ropar at Inter IIT Tech Meet 14.0 and Technoxian's National Micromouse Challenge.

Most recently, I designed a 5-stage pipelined MIPS32 processor — the layer just beneath the software I usually write.
